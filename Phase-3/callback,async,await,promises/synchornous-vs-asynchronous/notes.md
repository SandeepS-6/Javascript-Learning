what is synchornous and asynchronous ?

In javascript the code will exicute line by line naturally.but these asynchronous will exicute with a delay of time until previous code will exicute.

synchronous :- will exicute line by line and each task wait for the previous task to be finish.
One task at a time.

asynchronous :-JavaScript can start a task and continue executing other code without waiting for that task to finish.
JS does not wait; it continues other work.
